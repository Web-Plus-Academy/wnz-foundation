16-Week Curriculum

Your Learning Roadmap

A carefully crafted curriculum from fundamentals to production-ready applications.

Month-by-Month Breakdown

A progressive, four-month learning roadmap designed to take you from fundamentals to real-world application and community leadership.

1

Month 1

Web Foundations

Build a strong base in HTML, modern CSS, responsive layouts, accessibility best practices, and web standards. Learn how the web works and develop clean, structured, and scalable interfaces.

2

Month 2

JavaScript & Logic

Strengthen your problem-solving skills with core JavaScript concepts including variables, functions, control flow, DOM manipulation, asynchronous programming, and real-world logic building.

3

Month 3

ReactJS

Learn modern frontend development using ReactJS. Work with components, hooks, state management, routing, and API integration to build dynamic and scalable user interfaces.

4

Month 4

Capstone Project & Mentorship

Build a complete real-world project from scratch while receiving mentorship and code reviews. Transition from learner to mentor by supporting juniors and contributing actively to the community.
